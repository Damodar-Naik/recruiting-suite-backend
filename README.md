This is a backend for mockup of a Recruitment Suite

Stack :
- Nextjs, for frontend.
- NodeJs & express, for backend.
- Gemini, for LLM.
- AFFINDA, for Resume Parsing.
- postgres, for DB.

---------------------------------------------------------------------------
steps to setup the project loacally:

- clone the repository
- npm i
- npm run dev

env variables :
- PORT
- DB_USER
- DB_PASSWORD
- DB_NAME
- DB_HOST
- DB_PORT
- AFFINDA_API_KEY
- AFFINDA_WORKSPACE_ID
- GEMINI_API_KEY
- GEMINI_MODEL
- FRONTEND_URL
